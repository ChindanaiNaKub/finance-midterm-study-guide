### üìä TIME VALUE OF MONEY (TVM)

```
Simple Interest:        I = P √ó r √ó t

Compound Interest:      FV = PV(1 + r)^t
                       PV = FV / (1 + r)^t
                       PV = FV(1 + r)^(-t)  [Alternative form]

Rule of 72:            Years = 72 √∑ Interest Rate

Future Value Annuity:   FVA = PMT/r √ó [(1+r)^t - 1]
                       FVA = PMT √ó [(1+r)^t - 1] / r

Present Value Annuity:  PVA = PMT/r √ó [1 - 1/(1+r)^t]
                       PVA = PMT √ó [1 - 1/(1+r)^t] / r
```

**‚ö° Quick Check:**
- r = decimal (8% ‚Üí 0.08)
- Monthly: r√∑12, t√ó12
- FV = Future Value | PV = Present Value

### üí≥ TAX BRACKETS (Progressive Tax System)

| Taxable Income | Rate |
| --- | --- |
| 0 - 8,350 | 10% |

**Tax Calculation Steps:**
```
1. Gross Income
2. - Adjustments (Student loans, moving expenses)
3. = AGI (Adjusted Gross Income)
4. - Deductions (Standard OR Itemized - choose higher)
5. - Exemptions (3,650 √ó number of people)
6. = Taxable Income
7. ‚Üí Calculate tax (each bracket √ó rate ‚Üí sum)
8. - Tax Credits (dollar-for-dollar reduction)
9. = Tax Owed/Refund
```

**Standard Deduction:**
- Single: 5,700
- Married (Joint): 11,400

**Marginal vs Average:**
- Marginal = Rate of last dollar earned
- Average Tax Rate = Total Tax / Gross Income

**Tax Credits > Deductions:**
- Credit reduces tax directly: $1,000 credit = $1,000 savings
- Deduction reduces taxable income: $1,000 deduction √ó Tax Rate = savings

### üè¶ LOAN INTEREST METHODS

**1. Declining-Balance** (Mortgage, Credit Cards)
```
Interest calculated on remaining balance (decreases monthly)

Month 1:
Interest = Beginning Balance √ó Periodic Rate
Principal = Payment - Interest
Ending Balance = Beginning - Principal
```

**2. Add-On Method** (Car, Furniture)
```
I = P √ó R √ó T
Total Repay = P + I
Monthly Payment = Total / Number of Months
```

**3. Discount Method**
```
I = P √ó R √ó T
Amount Received = P - I (interest paid upfront)
Monthly Payment = P / Number of Months
```

**4. Monthly Payment Table**
```
Find from table: (Interest Rate, Number of Months)
‚Üí Get value per $1,000

Monthly Payment = (Table Value) √ó (Principal/1,000)
Finance Charge = (Payment √ó Months) - Principal
```

### üìä ECONOMIC INDICATORS
